3.1 Keyboard shortcut

  • C-x a : enter or leave TUI mode
  • C-x 1 : one window
  • C-x 2 : two windows
  • C-x o : 'other'
  • C-l : refresh

3.1.1 Navication

up, down, left, right, pgup, pgdown

3.1.2 Single key mode

  • C-x s : single key mode
  • (c) continue
  • (d) down
  • (f) finish
  • (n) next
  • (o) nexti (step over)
  • (q) exit single key mode
  • (r) run
  • (v) info locals
  • (w) where
